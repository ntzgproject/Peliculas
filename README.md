# Buscador de Peliculas

Buscador de peliculas inspirado en el trabajo final de "Codo a Codo" bootcamp de programacion. 

## Tecnologias usadas: 

- **HTML**
- **CSS**
- **Javascript (Vanilla)**
 
## Uso

Para usarlo solamente descarga el proyecto y ejecuta el archivo "Index.html". Se que no es lo correcto pero para fines practicos deje la API Key de la libreria para que puedas usarla ni bien lo descargues :wink:
(En el caso de no tener acceso simplemente entra a https://www.themoviedb.org y crea tu API Key).

## Descripcion:

- Previsualiza las peliculas del momento y en la seccion "aclamadas" las mas populares.
- Permite buscar peliculas y acceder a su informacion.
- Realize un registro y un login ficticio a modo ilustrativo.
- Es responsive para Desktop y Mobile.
- Utilice la meotodogia BEM y Mobile First.
